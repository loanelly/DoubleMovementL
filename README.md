# DoubleMovementL 🎮

Универсальный кастомный мини-контроллер на базе Arduino Nano для плавного перемещения и управления в любых компьютерных играх. 

Плата считывает наклоны аналогового джойстика и клик механической кнопки, отправляя сырые данные по USB. Софт на Python с удобным графическим интерфейсом (GUI) мгновенно переводит эти сигналы в нажатия клавиш на низком уровне Windows (DirectInput). Это обеспечивает совместимость с большинством современных игр и гарантирует нулевую задержку ввода.

## 🛠️ Компоненты проекта

* **Arduino Nano** (или Arduino Uno)
* **Аналоговый джойстик** (модуль XY-джойстика)
* **Механический переключатель** (кнопочный свитч под палец)
* **Соединительные провода**

### Схема подключения:
* **VERT (Ось Y)** ➡️ Пин **A0** на Arduino
* **HORIZ (Ось X)** ➡️ Пин **A1** на Arduino
* **SEL (Кнопка)** ➡️ Пин **D2** на Arduino
* **VCC** ➡️ **5V**
* **GND** ➡️ **GND**

## 🚀 Возможности софта

* **Автоматический поиск порта:** Программа сама сканирует систему, находит чипы связи (CH340/Arduino) и подключается в один клик.
* **Настройка мертвой зоны:** Ползунок регулировки люфта стика прямо в окне (чтобы персонаж не двигался сам, когда джойстик в покое).
* **Кастомные бинды:** Возможность «на лету» переназначить кнопки движения или действий (по умолчанию WASD + E) без перезапуска скрипта.
* **0 ms Input Lag:** Чтение данных порта вынесено в отдельный асинхронный поток, обеспечивая максимальный отклик в динамичных сценах.

## 💻 Инструкция по запуску

### 1. Прошивка платы
Загрузи код из папки проекта в свою плату **Arduino Nano** через Arduino IDE. Скорость Serial-порта в коде должна быть настроена на `115200`.

### 2. Подготовка окружения на ПК
Для работы Python-скрипта с графическим интерфейсом установи необходимые библиотеки через командную строку (CMD):
```bash
pip install pyserial pydirectinput
```

### 3. Старт
Запусти управляющую программу:
```bash
python PythonSoftware.py
```
В открывшемся окне нажми кнопку **"Старт"** и запускай любую игру!
