# DoubleMovementL
Геймпад Martoz на Arduino Nano — мини-контроллер для плавного бега (Double Movement) в Fortnite. Плата считывает наклоны джойстика и клик кнопки, отправляя данные по USB. Софт на Python с GUI-интерфейсом мгновенно эмулирует WASD+E на низком уровне Windows без задержек. Есть автопоиск COM-порта, смена клавиш и настройка мертвой зоны стика.
